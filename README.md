# Hero Academy

Hero Academy is a Flask-based educational game platform built for a gifted 10-year-old learner who needed something more engaging than standard math drills.

## Features

- Adaptive Math Arena with Beast Mode challenges
- Writing Quest with XP rewards
- Progress dashboard and leaderboard
- Boss Battle Mode with unlockable elite challenges
- SQLite save system for persistent progress
- Sound effects, animations, and reward feedback

## Tech Stack

Python · Flask · SQLite · HTML · CSS · JavaScript

## Why I Built It

I built Hero Academy for my grandson Carlo to make learning feel like an adventure instead of a worksheet. The goal was to combine academic challenge, game mechanics, and emotional motivation into one playful learning experience.

## What I Learned

- Building Flask routes and templates
- Managing persistent user progress with SQLite
- Designing interactive front-end behavior with JavaScript
- Turning a real user need into product features
- Balancing fun, feedback, and learning outcomes
