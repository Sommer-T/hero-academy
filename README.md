## 🧠 Core Features

### 🧠 Adaptive Math Arena
- Dynamically generated high-level math problems  
- “Beast Mode” for advanced learners  
- Speed + accuracy challenge system  
- Designed to **push beyond standard curriculum limits**

---

### ✍🏾 Writing Quest
- Guided prompts to strengthen written expression  
- XP rewards for completion  
- Bridges gap between thinking and articulation  

---

### 🏆 Progress Dashboard
- Tracks XP, levels, and performance  
- Competitive leaderboard system  
- Reinforces consistency and growth  

---

### 👹 Boss Battle Mode
- Unlockable elite challenges  
- Milestone-based progression  
- High-difficulty encounters to test mastery  

---

### 💾 Persistent Progress System
- SQLite database integration  
- Player progress saved across sessions  

---

### 🎆 Game Feedback System
- Sound effects  
- Confetti animations  
- Victory feedback loops  
- Designed to make success **felt**, not just seen  

---

## 🛠 Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite
